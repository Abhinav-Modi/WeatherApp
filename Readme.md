# Weather App 

Made Using Reactjs and ExpressJs

## 
Go to client folder and install packages
```bash
 cd client
```
```bash
 npm i
```
Go to server folder and install packages
```bash
 cd server
```

```bash
 npm i

```
Now in the server folder 
 ```bash
 npm start
```
